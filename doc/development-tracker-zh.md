## 页面 QA 清单

已通过实现审查确认：

- 头部导航有四个标签。
- 移动端底部标签对应主功能区。
- 搜索表单提交 `q`。
- 搜索页能从 `q` 初始化、获取更新结果、客户端筛选、高亮匹配并分页。
- 状态胶囊链接到复习册。
- 推荐诗详情链接指向 `/poems/[id]`。
- 继续学习入口指向 `/poems/[id]/test`。
- 复习册入口指向 `/review-book`。
- 诗词详情展示插图、元数据、音频、诗句、注释、译文和赏析。
- 默写测试通过 API 提交答案并更新学习进度。
- 复习册移除会将诗移回 `learning`。
- 进度百分比由真实数据驱动。
- 最近学习不是假数据兜底。
- 推荐内容在客户端轮换。

仍需浏览器/手动确认：

- 首页、搜索、详情、复习和测试页面的最终桌面间距。
- 移动端底部标签的最终手感。
- 推荐轮换的视觉状态。
- 常见真实查询的搜索结果相关性。
- 有/无生成音频时的音频行为。

- 运行 `npm run prepare:v1`。
